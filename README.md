# 📝 Doc Studio

> A retro-inspired document creation platform for developers and professionals.

---

## ✨ Overview

**Doc Studio** is a modern web application that allows you to create and manage different types of documents in one place, including:

- 📄 Software Documentation  
- 🔗 API Documentation  
- 📃 Resumes  
- ✉️ Emails  
- 🗒️ Meeting Notes  
- 📚 And more...

Designed with a **retro 90’s aesthetic**, Doc Studio delivers a clean, distraction-free writing experience with a nostalgic touch.

---

## 🎨 Features

- ⚡ Fast and lightweight  
- 🧠 Minimal, distraction-free editor  
- 🎯 Multi-purpose document creation  
- 💾 Local development support  
- 🎨 Unique retro UI design  

---

## 🛠️ Tech Stack

- **Frontend:** Vite + JavaScript  
- **Editor:** VS Code (recommended)  
- **Runtime:** Node.js  

---

## 🚀 Getting Started

### 📌 Prerequisites

Make sure you have the following installed:

- 🧑‍💻 A code editor (VS Code recommended)  
- 🟢 [Node.js (LTS Version)](https://nodejs.org/en)  
- 🔧 Git  

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/zroxcoder/Doc-studio.git

Navigate into the project directory:

cd Doc-studio

Install dependencies:

npm install
▶️ Run Locally

Start the development server:

npm run dev

The app will be available at:

http://localhost:5173

⚠️ If port 5173 is in use, Vite will automatically switch to another port (e.g., 5174).

🌐 Access on Local Network

To share the app across devices on the same network:

Option 1: VS Code Live Server
Install Live Server extension from VS Code Marketplace
Launch the project using Live Server
Option 2: Vite Network Exposure

Run:

npm run dev -- --host

This will expose your local server to your network.
